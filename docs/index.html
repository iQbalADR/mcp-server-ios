Quick Start

Prerequisites: Xcode 26.x, Ollama, Python 3.10+

  1. Run Setup Script

    ./scripts/setup.sh

    Creates virtual environment, installs dependencies, pulls models.

  2. Create Custom Models

    ./scripts/create_models.sh

    Creates specialized iOS Swift Architect models.

  3. Ingest Your Project

    python scripts/ingest_project.py /path/to/your/project

    Adds your codebase to the vector database for context.

  4. Start the Server

    ./scripts/start_server.sh

    Starts the MCP server with debugging enabled.

  5. Test the Server

    npx @anthropic-ai/mcp-inspector ./venv/bin/python -m server.mcp_server

    Or run directly: ./venv/bin/python -m server.mcp_server

Note: Native Xcode MCP integration requires a future Xcode version with MCP support. For now, use the MCP Inspector or run the server directly for testing.

Available Tools

MCP tools exposed to Xcode's AI Assistant:

Tool Description
generate_code Generate code from natural language with RAG context
explain_code Explain what code does in plain English
fix_code Fix broken code based on error messages
ask_with_context Ask any question with codebase context
add_to_memory Add code/docs/notes to persistent memory
search_memory Search through your code notebook
get_memory_stats Get vector database statistics
clear_memory Clear a memory collection

Configuration

Edit config.yaml to customize the server:

Ollama Settings

ollama:
  base_url: "http://localhost:11434"
  chat_model: "ios-swift-architect"
  embedding_model: "nomic-embed-text"
  temperature: 0.1
  max_tokens: 4096

LanceDB Settings

lancedb:
  db_path: "./data/lancedb"
  max_results: 10
  similarity_threshold: 0.7

Debug Settings

debug:
  enabled: true
  save_requests: true
  request_log: "./logs/requests.jsonl"
